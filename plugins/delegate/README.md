# delegate

Claude Code plugin for delegation workflows, voice processing, email, messages, calendar, and more.

## Commands

- **delegate** — Process delegation.md task queue (interactive)
- **delegate-auto** — Autonomous delegation processor (for scheduled runs)
- **voice** — Route voice notes from voice.md to tasks, delegation, and project files
- **morning** — Morning brief with tasks, calendar, and email context
- **weekly** — Weekly review as chief of staff
- **calendar** — View and query calendar events via icalBuddy (Mac only)
- **mail** — Read email via Mac Mail app (Mac only)
- **messages** — Access messages across platforms via Beeper Desktop
- **drive** — Browse, search, and download from Google Drive via rclone
- **push** — Git commit and push with auto-generated message
- **editors** — Multi-perspective writing feedback using parallel subagents
- **meeting** — Search and summarize meeting notes
- **setup-transcription** — Set up automatic voice memo transcription

## Platform Requirements

- **macOS**: calendar, mail, voice memos transcription
- **All platforms**: delegation, voice routing, drive, messages, push, editors, meeting, weekly

## Bundled Scripts

Scripts for mail, messages, drive setup, and transcription are included in `scripts/`. No manual downloads needed.
